# 📝 13. GUI To-Do Менеджері (Java Swing)

## 🎯 Жобаның мақсаты
Java тілінде Swing немесе JavaFX арқылы графикалық интерфейсі бар To-Do (тапсырмалар тізімі) қосымшасын жасау.  
Бұл жоба арқылы GUI компоненттерімен жұмыс істеу және оқиға (event) өңдеуді үйрену.

---

## 📌 Жобаның сипаттамасы
Бұл қосымша қолданушыға:
- Тапсырма қосу
- Тапсырманы өшіру
- Орындалғанын белгілеу

мүмкіндіктерін береді.

Қолданылатын GUI компоненттері:
- JFrame (негізгі терезе)
- JButton (батырмалар)
- JTextField (мәтін енгізу)
- JList (тапсырмалар тізімі)

---

## 🧠 Жобаны іске асыру жоспары
1. JFrame терезесін құру  
2. JTextField қосу (тапсырма енгізу үшін)  
3. “Add” батырмасын жасау  
4. JList арқылы тізім көрсету  
5. Delete батырмасын қосу  
6. Done батырмасы арқылы орындалғанын белгілеу  
7. ActionListener арқылы оқиғаларды басқару  

---

## ❓ Сұрақтар және жауаптар

### 1. JFrame деген не?
JFrame — Java Swing ішіндегі негізгі терезе.  
Барлық GUI компоненттері осы терезе ішінде орналасады.  
Бұл қосымшаның басты экраны болып табылады.

---

### 2. ActionListener қалай жұмыс істейді?
ActionListener — батырма басылған кезде орындалатын әрекетті анықтайды.  
actionPerformed() әдісі арқылы іске қосылады.

Мысалы:
- Add батырмасы → тапсырма қосады  
- Delete батырмасы → тапсырма өшіреді  

---

### 3. GUI мен логиканы қалай бөлуге болады?
GUI мен логиканы бөлу үшін:
- GUI → интерфейс (батырма, текст, тізім)
- Logic → деректермен жұмыс (қосу, өшіру, өңдеу)

Көбіне бөлек класстар қолданылады:
- UI class → интерфейс  
- Service class → логика  

Бұл кодты таза және түсінікті етеді.

## 📌 Қорытынды
Бұл жоба Java Swing арқылы графикалық интерфейс жасауды үйретеді.  
JFrame, JButton, ActionListener сияқты негізгі компоненттермен жұмыс істеу дағдысы қалыптасады.  
Нәтижесінде толық жұмыс істейтін To-Do қосымша жасалады.
